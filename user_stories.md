# User Stories

1. As a user, I want to register an account so that I can use the app.
2. As a user, I want to log in so that I can access my account.
3. As a user, I want to view the home screen so that I can see available items.
4. As a user, I want to view details of an item so that I can understand it better.
5. As a user, I want to save items as favorites so that I can access them later.
6. As a user, I want my data to be stored locally so that it persists after closing the app.
7. As a user, I want to fetch data from an API so that I see updated information.
8. As a user, I want to access settings so that I can customize the app.
9. As a user, I want to receive notifications so that I stay updated.
